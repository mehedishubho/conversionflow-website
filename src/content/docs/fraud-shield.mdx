Fraud Shield helps store teams spot risky COD orders before sending parcels to courier partners.

## Detection Rules

WooBooster can flag repeated phone numbers, high return history, suspicious address patterns, and mismatched customer details.

## Configure Rules

1. Open **WooBooster > Fraud Shield**.
2. Enable return history checks.
3. Set the number of returns that should trigger a warning.
4. Add trusted customers to the whitelist when needed.
5. Review blocked orders before dispatch.

## Order Scoring

Each flagged order receives a visible risk state in the order admin screen. Your team can still manually approve an order after customer confirmation.

## Whitelist and Blacklist

Use the whitelist for trusted repeat customers. Use the blacklist for phone numbers or addresses that repeatedly cause failed deliveries.
