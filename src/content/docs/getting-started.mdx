WooBooster is a WooCommerce automation plugin for Bangladeshi store owners. This guide walks through installation, activation, and the first configuration pass.

## Requirements

| Requirement | Minimum |
| --- | --- |
| WordPress | 6.0 or newer |
| WooCommerce | 5.0 or newer |
| PHP | 7.4 or newer |
| SSL | Required for secure API calls |

## Install the Plugin

1. Download the WooBooster ZIP file from your account.
2. Open **Plugins > Add New** in WordPress.
3. Choose **Upload Plugin** and select the ZIP file.
4. Click **Install Now**, then **Activate**.

## Activate Your License

Open **WooBooster > License**, paste your license key, and click **Activate License**. License validation enables updates, support access, and module activation for your purchased tier.

## Initial Configuration

Start with your store details, support email, and preferred currency. Then connect courier APIs, configure fraud rules, and test one order before enabling automation for all new orders.
