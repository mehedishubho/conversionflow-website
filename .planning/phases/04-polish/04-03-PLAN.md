## Tasks

### Task 1: Audit and optimize font weights

**Files Modified:** `src/app/layout.tsx`

**Action:**
Review font weight usage across all components. The Syne font loads weights 400, 600, 700, 800. Check if weight 800 is actually used in the codebase. If only headings use 600, 700, 800 — keep those. Remove 400 if Syne is only used for headings (DM Sans handles body text).

Current config:
```tsx
const syne = Syne({
  variable: "--font-syne",
  subsets: ["latin"],
  weight: ["400", "600", "700", "800"],
});
```

After audit, update to only include weights actually referenced in CSS/className. The `.sec-title` uses `font-extrabold` (800), `.step-t` uses `font-extrabold` (800), `.eyebrow` uses `font-extrabold` (800). Weight 400 (`font-normal`) is not explicitly used for Syne. Remove it:

```tsx
const syne = Syne({
  variable: "--font-syne",
  subsets: ["latin"],
  weight: ["600", "700", "800"],
});
```

**Acceptance Criteria:**
- `src/app/layout.tsx` Syne font config does NOT contain `"400"`
- `src/app/layout.tsx` Syne font config contains `"800"`
