## Tasks

### Task 1: Update root layout metadata with OG base tags and favicon config

**Files Modified:** `src/app/layout.tsx`

**Action:**
Update the `metadata` export in `src/app/layout.tsx` to include:
- `metadataBase` set to `https://woobooster.com`
- `title.template` pattern: `%s — WooBooster`
- `title.default`: `WooBooster — WooCommerce Automation for Bangladesh`
- `description`: Keep existing
- `openGraph` with `siteName: "WooBooster"`, `locale: "en_US"`, `type: "website"`
- `icons` pointing to `/favicon.svg`
- `twitter:card` set to `summary_large_image`

```tsx
export const metadata: Metadata = {
  metadataBase: new URL("https://woobooster.com"),
  title: {
    default: "WooBooster — WooCommerce Automation for Bangladesh",
    template: "%s — WooBooster",
  },
  description: "Automated courier sync, Meta CAPI tracking, and Fraud Shield for Bangladeshi WooCommerce stores.",
  openGraph: {
    siteName: "WooBooster",
    locale: "en_US",
    type: "website",
  },
  twitter: {
    card: "summary_large_image",
  },
  icons: {
    icon: "/favicon.svg",
  },
};
```

**Acceptance Criteria:**
- `src/app/layout.tsx` contains `metadataBase: new URL("https://woobooster.com")`
- `src/app/layout.tsx` contains `title.template`
- `src/app/layout.tsx` contains `openGraph`
- `src/app/layout.tsx` contains `icons`
