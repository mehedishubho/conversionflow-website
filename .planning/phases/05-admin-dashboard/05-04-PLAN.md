    - Define interface `CSVColumn { header: string; accessor: (row: Record<string, unknown>) => string }`.
    - Export function `exportToCSV(columns: CSVColumn[], rows: Record<string, unknown>[], filename: string)`:
      - Build header row: `columns.map(c => '"' + c.header + '"').join(",")`.
      - Build data rows: for each row, map columns to escaped values. Escape quotes by doubling them: `val.replace(/"/g, '""')`. Wrap each value in quotes.
      - Join all rows with `\n`.
      - Prepend BOM character `"﻿"` for Excel UTF-8 compatibility.
      - Create Blob: `new Blob([csv], { type: "text/csv;charset=utf-8;" })`.
      - Create object URL, create temporary `<a>` element, set `href` and `download` attribute to `${filename}-${YYYY-MM-DD}.csv`.
      - Append to body, click, remove from body, revoke object URL.

    - Call requireAdmin().
    - Default page=1, limit=25.
    - Build WHERE conditions:
      - If eventType is provided and not "all": filter by `auditLogs.action LIKE '${eventType}%'` using `sql`${auditLogs.action} LIKE ${eventType + '%'}``.     - Query count for pagination: `SELECT COUNT(*) FROM auditLogs WHERE conditions`.
    - Query data: `SELECT auditLogs.*, user.name as actorName FROM auditLogs LEFT JOIN user ON auditLogs.actorId = user.id WHERE conditions ORDER BY auditLogs.createdAt DESC LIMIT limit OFFSET (page-1)*limit`.
    - Return `{ events, totalCount, totalPages: Math.ceil(totalCount / limit), currentPage: page }`.

    - Same as getFullActivity but without pagination — returns all matching events for CSV export.
