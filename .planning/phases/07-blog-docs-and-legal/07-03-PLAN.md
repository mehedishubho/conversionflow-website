**src/components/docs/TableOfContents.tsx** -- Client component (needs IntersectionObserver). Mark with `"use client"` at first line.

Props: none (extracts headings from DOM after mount).

Implementation:
1. State: `headings: { id: string; text: string; level: number }[]` and `activeId: string | null`
2. useEffect on mount: query `article.prose h2[id], article.prose h3[id]` from the DOM. Map each to `{ id: el.id, text: el.textContent || "", level: parseInt(el.tagName[1]) }`. Store in headings state.
3. Second useEffect (depends on headings): create `IntersectionObserver` with `{ rootMargin: "-80px 0px -70% 0px" }`. Observe each heading element. On intersection, set `activeId` to the `id` of the first intersecting heading. Return cleanup to disconnect observer.
4. Render: if headings.length === 0, return null. Otherwise render a nav element:
   - Container: `hidden xl:block` (only visible on extra-large screens)
   - Each heading: `<a href="#${id}">` with:
     - Default: `text-[13px] font-semibold text-muted transition-colors duration-150 hover:text-text2 block py-0.5`
     - Active (id === activeId): `text-accent font-bold`
     - h3 items (level === 3): add `pl-4` indentation
